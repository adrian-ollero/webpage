    Installation¶

    Note

    Before using this theme you have to install nodejs

    1. Install a themeâ€™s dependencies from your command line via npm. type npm install
    2. Run a npm build to packing using webpack for more detail visit official documentation

    How to running local server¶

    after install dependencies, now you can running a local server without instalation plugin, in command line type npm run serve

    How to running local server with plugin [Alternative]¶

    you can also use alternative way using plugin from code editor visual studio code

    1. install the code editor Visual Studio Code (download similar with your operating system)
    2. install the plugin Live Server

    Method 1¶

    1. Open a project and click to Go Live from the status bar to turn the server on/off.

    Method 2¶

    1. Open a HTML file and right-click on the editor and click on Open with Live Server.

    Method 3¶

    1. Hit alt+L, alt+O to Open the Server and alt+L, alt+C to Stop the server (You can change the shortcut form keybinding). [On MAC, cmd+L, cmd+O and cmd+L, cmd+C]
